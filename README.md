# Card Game

Currently a work in progress - will refactor and clean up code later.

To run:

Clone this repository.

`$ npm init`  
`$ npm install`  
`$ npm run server`

To start on heroku:
`$ heroku ps:scale web=1`

To stop:
`$ heroku ps:stop web.1`

To restart server (due to crash or new game):
`$ heroku ps:restart`

To check current status:
`$ heroku ps`

To run locally:
$ heroku local web
